## 从前世Bro吸收的经验

### Git/备份
- GitHub push 有时超时（443端口不稳定）
- 本地领先时需要 rebase 后再 push

### API配置
- Minimax 正确模型：`MiniMax-M2.5-highspeed`
- 端点：`https://api.minimaxi.com/v1`

### 数据采集
- SHFE: AkShare 可用，数据到2026-02-13
- DCE: 有瑞数反爬，浏览器模拟也困难
- CZCE/CFFEX: 需要其他数据源

### 网络工具
- web_search 需要 Brave API key
- 无头浏览器 vs Chrome relay 选择
