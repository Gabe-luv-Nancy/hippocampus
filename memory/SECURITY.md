# 安全配置与权限边界

## 部署环境
- 运行在 Gabe 的私人笔记本电脑 (Windows) 上
- 通过 WSL 安装 Ubuntu
- 我的部署与他的很多文件在同一磁盘下

## 安全权限规则

### 🔴 禁止操作
1. **Windows 系统文件** — 不要直接操作 Windows 系统中的文件
2. **WSL 关键配置文件** — 不要直接操作影响我本身进程或影响正常回复的配置文件

### 🟢 可以操作
- 其他计划文件都可以随意更改
- 工程合作文件，只要有良好备份，可以任意更改

### ⚠️ 模糊地带
- 先问再做：不确定的文件先确认
