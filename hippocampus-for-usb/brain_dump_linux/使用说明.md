## 快速入门

### 第一步：打开软件

**方式一（推荐）：双击 `1-本地版.bat`**
> 自动选择最佳方式，浏览器弹出 Web 界面

**方式二：双击 `start-streamlit.bat`**
> 单独启动 Streamlit Web UI

**方式三：双击 `Hippocampus.exe`**
> PyQt6 桌面版（如已通过 build.bat 构建）

## 各功能详解

### 📊 仪表盘

查看当前已备份的记忆文件统计：
- 共有多少文件
- 总大小
- 抓取过几次
- 有哪些 AI 来源

### 📂 文件浏览器

浏览 U盘 里已备份的所有记忆文件。

**使用方法：**
1. 在左侧选择「文件浏览器」
2. 点击「筛选来源」下拉框，选择某个 AI 来源
3. 点击文件条目可以展开预览内容

### 🖥️ 主机检测（本地版专属）

这是核心功能——把电脑上的记忆文件备份到 U盘。

**操作步骤：**

1. **检测 AI 工具** — 点击「🔍 检测 AI 工具」按钮
   > 软件会扫描电脑，列出已安装的 AI 工具

2. **选择要扫描的来源** — 在下拉框中勾选要扫描的 AI 工具
   > 例如：只勾选 OpenClaw

3. **扫描记忆文件** — 点击「📡 扫描记忆文件」
   > 显示电脑上找到的所有 .md 记忆文件

4. **勾选要备份的文件** — 展开各来源，勾选需要的文件

5. **备份到目标** — 点击「💾 备份到目标」
   > 文件复制到 U盘 + 写入数据库

### 📜 抓取历史

查看历次备份记录：时间、文件数量、来自哪台电脑。

### ⚙️ 设置

- **切换模式**：本地版 / 磁盘版
- 本地版：扫描电脑 → 备份到目录或 U盘
- 磁盘版（U盘版）：只读浏览 U盘已有内容

## 备份目标说明

软件顶部会显示当前备份目标：

| 状态 | 颜色 | 含义 |
|------|------|------|
| 🟢 BrainDump 已连接 | 绿色 | 检测到 U盘，可以备份 |
| 🟡 外部磁盘 | 橙色 | 是 U盘 但不是 BrainDump |
| 🔴 未连接 | 红色 | 未检测到任何磁盘 |

**手动切换目标**：点击「备份目标」下拉框，选择其他目录或 U盘。

## 常见问题

### Q: 提示"未检测到外部磁盘"
**A**: 确认 U盘 已插入电脑且被 Windows 识别。可以先在文件管理器里确认 U盘 能打开。

### Q: 备份时提示"数据库不存在"
**A**: 运行 `db\init_db.py` 初始化数据库。

### Q: 想备份到本地目录而不是 U盘
**A**: 点击「备份目标」下拉框 → 选择目录 → 手动指定一个本地文件夹。

### Q: Streamlit 启动失败
**A**: 确认已安装 Python（`python --version`），然后运行：
```bash
pip install streamlit
```
